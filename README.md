### Q&A

##### 1. 为什么要搞这个编辑器？

因为 B 站上传封面时，那个调节图片大小的框 无论如何操作，都会给我切掉边距，很恼火。我不得不每次做封面时 都留一块边距出来，很繁琐。

##### 2. 为什么要用拖拉拽的方式？

潮流。

而且用来做这个需求感觉挺合适的，也挺简单的。
